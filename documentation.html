Getting Started

Welcome to the MediaPipe to Blender documentation. This guide will help you get started with using the add-on to create real-time animations in Blender.

Installation

Follow these steps to install the MediaPipe to Blender add-on:

  1. Download the add-on ZIP file from the download page.
  2. Open Blender and go to Edit > Preferences > Add-ons.
  3. Click "Install..." and select the downloaded ZIP file.
  4. Enable the add-on by checking the box next to "Animation: MediaPipe Motion Capture".

The first time you use the add-on, it will automatically install required Python packages. This requires an internet connection and may take a few minutes.

Note

If you encounter any issues during installation, please check the troubleshooting section or visit the support page.

Interface Overview

The add-on adds a new panel to the 3D View sidebar. To access it:

  1. In the 3D View, press N to open the sidebar.
  2. Select the "MediaPipe MoCap" tab.

The interface is divided into three main panels:

  • Bone Mapping: Configure how MediaPipe landmarks map to armature bones.
  • Animation Settings: Control animation parameters and recording.
  • Status: Monitor connection status and statistics.

The MediaPipe to Blender interface in Blender's sidebar

Quick Start Guide

To quickly get started with MediaPipe to Blender:

  1. Open a Blender file with an armature you want to animate.
  2. Make sure the armature is in rest position.
  3. In the Bone Mapping panel, select your armature from the dropdown.
  4. Choose a mapping mode (Auto, Preset, or Manual).
  5. Click "Create Mapping" to generate the initial mapping.
  6. In the Status panel, click "Connect" to start the MediaPipe connection.
  7. In the Animation Settings panel, click "Start Recording" to begin animating.

Your armature should now move in response to your movements captured by the webcam.

Bone Mapping

Bone mapping is the process of connecting MediaPipe landmarks to Blender armature bones. This section explains the different mapping options available in the add-on.

Mapping Modes

The add-on provides three ways to map MediaPipe landmarks to Blender armature bones:

  • Auto Mapping: Automatically maps landmarks based on bone names.
  • Preset Mapping: Uses predefined mappings for popular rigs (Rigify, Mixamo).
  • Manual Mapping: Allows you to manually map each bone to specific landmarks.

To select a mapping mode, use the dropdown menu in the Bone Mapping panel.

Selecting a mapping mode in the Bone Mapping panel

Manual Mapping Editor

The manual mapping editor allows you to create custom mappings for each bone. To use it:

  1. Select "Manual" mapping mode.
  2. Click the "+" button to add a new mapping.
  3. Select the bone you want to map:
    • Type the bone name, or
    • Click the eyedropper icon and select a bone in the 3D view.
  4. Choose the landmark type:
    • Face: 468 facial landmarks
    • Hand: 21 landmarks per hand
    • Pose: 33 body landmarks
  5. Set the landmark index (refer to the landmark reference for specific indices).
  6. For hand landmarks, select which hand (Left or Right).
  7. Click "Apply Bone Mappings" to apply your changes.

The manual mapping editor interface

Preset Mappings

The add-on includes preset mappings for popular rigs:

  • Rigify: For Blender's built-in Rigify system.
  • Mixamo: For characters from Mixamo.

To use a preset mapping:

  1. Select "Preset" mapping mode.
  2. Choose the preset from the dropdown menu.
  3. Click "Create Mapping" to apply the preset.

You can also save your custom mappings as presets for future use:

  1. Create your mapping using the manual mapping editor.
  2. Click "Save as Preset" in the Bone Mapping panel.
  3. Enter a name for your preset and click "Save".

Animation Settings

The Animation Settings panel provides controls for how the captured motion is applied to your armature.

Recording Controls

The recording controls allow you to start and stop animation recording:

  • Start Recording: Begin applying motion capture data to the armature.
  • Stop Recording: Stop applying motion capture data.
  • Auto Keyframe: When enabled, automatically inserts keyframes during recording.
  • Keyframe Interval: Sets how often keyframes are inserted (in frames).

Tip

You can use Blender's standard animation tools to edit the recorded animation after capturing.
