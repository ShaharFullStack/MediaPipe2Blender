Installation Instructions

1

Download the Add-on

Click the download button above to get the latest version of MediaPipe to Blender.

2

Open Blender Preferences

In Blender, go to Edit > Preferences > Add-ons.

3

Install Add-on from File

Click "Install..." and select the downloaded ZIP file.

4

Enable the Add-on

Check the box next to "Animation: MediaPipe Motion Capture" to enable the add-on.

5

First-Time Setup

The first time you use the add-on, it will automatically install required Python packages. This requires an internet connection and may take a few minutes.

Note: For detailed installation instructions and troubleshooting, please refer to the documentation.

System Requirements

Software Requirements

  • Blender: Version 2.93 or newer (Blender 4.0+ recommended)
  • Python: Version 3.7 or newer (included with Blender)
  • Operating System: Windows 10/11, macOS 10.15+, or Linux

Hardware Requirements

  • Webcam: Any standard webcam or video input device
  • Processor: Intel Core i5 or equivalent (i7 recommended)
  • RAM: 4GB minimum (8GB+ recommended)
  • Graphics: Dedicated GPU recommended for better performance
